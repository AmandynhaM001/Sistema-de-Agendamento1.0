Novas Funcionalidades e Regras de Agendamento 1.1
As seguintes regras foram implementadas para garantir um agendamento mais organizado e eficiente:

1. Agendamento por Departamento/Serviço
Implementado o suporte para agendamentos específicos, permitindo que os usuários escolham o Departamento ou Tipo de Serviço desejado.

Isso garante que cada agendamento seja direcionado corretamente.

2. Prevenção de Duplicação de Horário
O sistema agora valida a disponibilidade do horário com base na agenda operacional, evitando que o mesmo horário de funcionamento seja agendado duas vezes (conflito de horários).

Regra: Não será mais possível marcar um horário que já esteja ocupado.

3. Restrição de Finais de Semana
O sistema de agendamento está restrito aos dias úteis.
